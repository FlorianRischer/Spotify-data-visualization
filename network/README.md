# Musical Brain Activity

SvelteKit + TypeScript Projekt für den "Musical Brain Activity" Neural Network Graph.

## Setup
1. Abhängigkeiten installieren:
   ```bash
   npm install
   ```
2. Entwicklung starten:
   ```bash
   npm run dev
   ```
3. Checks:
   ```bash
   npm run check
   ```

## Struktur
- src/lib/graph: Graph-Kern (Builder, Layout, Interaktion, Types)
- src/routes: Svelte-Seiten (aktuell Platzhalter in +page.svelte)

Weitere Implementierung: Canvas/WebGL Renderer, UI/Controls, Stores.
